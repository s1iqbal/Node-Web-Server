## Installation

OS X/Linux/Windows:

```sh
npm install
npm start | node server.js
```


## Usage example

The port is defined as 3000 initially. After starting the server, you can access the site using localhost:3000

## Development setup
Front End: HandlebarsJs, CSS6
Back End: ExpressJs
Runtime: Node
